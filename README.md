# 🖥️ DSK Delivery System - Admin Console

A comprehensive admin dashboard template providing analytics, order dispatch logs, driver tracking maps, and user feedback logs for the DSK Delivery ecosystem.

## 🚀 Features
- **Active Orders Panel**: Live list of pending and assigned delivery orders.
- **Driver Dispatch**: Assign deliveries to active drivers based on regional proximity.
- **Database Management UI**: CRUD tools for orders, drivers, and stores.

## 🛠️ Tech Stack
- HTML5, CSS3, JavaScript (ES6)
- Bootstrap / Tailwind CSS
